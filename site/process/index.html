Process

1. Process Manager

List Process [^1]

# display Linux processes
top

# list process
ps -aux

# sort by CPU usage
top -o %CPU

# sort by memory usage
top -o %MEM
ps aux --sort -rss
top # then press M

# count process
ps -eaf | awk '{print $8}' | sort | uniq -c | sort -nr

Kill Process

# kill process by name
pkill firefox

2. Screen

# create new screen
screen -S [screen_name]

# list screen
screen -ls

# exit from screen
Ctrl-a d

# reattach a session
screen -r [screen_pid]

# delete a screen
screen -X -S [screen_pid] quit

[^1]: Sorting down processes by memory usage
